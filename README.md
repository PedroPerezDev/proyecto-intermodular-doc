# TaskFlow

## Descripción

**TaskFlow** es una aplicación web diseñada para la gestión de tareas de forma sencilla y colaborativa.  
Permite a los usuarios crear, organizar y completar tareas, facilitando el trabajo en equipo y el seguimiento del progreso.



---

## Tecnologías utilizadas

Las tecnologías empleadas en este proyecto son:

- HTML5  
- CSS3  
- JavaScript  
- Git  
- GitHub
