##  Descripción del proyecto

**TaskFlow** es una aplicación web orientada a la gestión de tareas de forma colaborativa.  
Su objetivo es facilitar la organización del trabajo diario y mejorar la productividad de equipos pequeños.

![Ejemplo visual de la aplicación](img/logos2.jpg)

Este proyecto se desarrolla como parte de una práctica académica para aprender a documentar proyectos software utilizando **Markdown** y **GitHub Pages**.

---

##  Objetivos del proyecto

- Organizar tareas de forma clara
- Facilitar el trabajo colaborativo
- Aplicar buenas prácticas de documentación técnica
- Publicar documentación en formato web
