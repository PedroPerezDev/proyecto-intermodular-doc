# Uso de TaskFlow

![Pantalla principal](img/logos1.jpg)

---

## Casos de uso principales

**TaskFlow** permite a los usuarios gestionar sus tareas de manera sencilla. Algunos casos de uso típicos son:

1. **Crear una tarea**
   - El usuario añade un título y una descripción
   - Asigna fecha de entrega y prioridad
2. **Marcar tareas completadas**
   - El usuario selecciona la tarea y la marca como finalizada
3. **Organizar tareas por categoría**
   - Las tareas se pueden agrupar por proyecto o tipo
4. **Consultar progreso del equipo**
   - Los usuarios pueden ver el estado de las tareas asignadas a otros miembros

---

## Usuarios tipo

- **Estudiantes** que quieren organizar sus tareas y proyectos académicos
- **Equipos de trabajo pequeños** que necesitan coordinación diaria
- **Personas individuales** que desean gestionar su productividad
